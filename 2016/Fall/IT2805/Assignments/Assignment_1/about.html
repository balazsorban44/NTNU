Questions

  1. The server stores the necessary files for the webpage, and runs the server-sided (backend) tasks, like database managing, and other scripts.
  2. The client recieves the files from the server, and then renders it to the user's screen, usually in a graphical web browser.
  3. An IP address is a unique identifier for every single device which is connected to the internet. The devices on the Internet find each other with the help of this Internet Protocoll.
  4. The DNS, or Domain Name System is a system, which matches the IP addresses - which are hard to remember for humans - with a more recognisable (domain) name, like www.google.com
  5. The Internet is the largest network on Earth, connecting billions of devices (IoT - Internet of Things) together, while WWW or W3 - which stands for World Wide Web - is the set of standards/rules for websites supervised by W3C. If one considering to make a website, it should follow the W3C guidelines.
  6. HTML stands for Hyper Text Markup Language, and is a markup language. It is often called the "skeleton of the website", because it is where the content is. HTML can serve as a website per se, while CSS and JavaScript cannot. They only give aesthetics and interaction to the page.
